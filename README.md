# AI Orchestration

AIオーケストレーションツールのプロジェクトです。

## プロジェクト構造

```
.
├── orchestration/    # メインのオーケストレーションロジック
├── llm/             # LLM関連の機能
├── cui/             # CUIインターフェース
├── schemas/         # データスキーマ定義
└── tests/           # テストコード
```

## 必要条件

- Python 3.12以上
- Poetry

## セットアップ

1. リポジトリをクローン
```bash
git clone [repository-url]
cd ai-orchestration
```

2. 依存関係のインストール
```bash
poetry install
```

3. 開発環境のセットアップ
```bash
poetry shell
```
