## 指示

以下のポイントに注意して、個々のサブタスク結果を一貫性のあるコードソリューションに統合してください:

1. コードの一貫性と相互運用性を確保する
2. 命名規則やコーディングスタイルを統一する
3. 共通の機能や重複を最適化する
4. 適切なモジュール構造とアーキテクチャを形成する
5. 必要に応じてドキュメントやコメントを追加する

## 出力形式

以下のJSON形式で統合結果を出力してください:

```json
{
  "integrated_content": "統合されたコード",
  "format": "code",
  "language": "python",
  "metadata": {
    "architecture": "システムアーキテクチャの説明",
    "main_components": ["コンポーネント1", "コンポーネント2"],
    "usage_instructions": "使用方法の説明",
    "dependencies": ["依存ライブラリ1", "依存ライブラリ2"]
  }
}
```

コードを統合してください。
