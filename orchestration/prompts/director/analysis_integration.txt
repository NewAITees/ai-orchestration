## 指示

以下のポイントに注意して、個々のサブタスク結果を一貫性のある分析報告書に統合してください:

1. 各分析結果の関連性と整合性を確保する
2. 重要な発見や洞察を強調する
3. データと分析結果を論理的に構成する
4. 全体の結論と推奨事項を導き出す
5. 必要に応じて図表や参考文献を整理する

## 出力形式

以下のJSON形式で統合結果を出力してください:

```json
{
  "integrated_content": "統合された分析報告書",
  "format": "analysis_report",
  "metadata": {
    "key_findings": ["主要な発見1", "主要な発見2"],
    "methodology": "使用した分析手法の説明",
    "data_sources": ["データソース1", "データソース2"],
    "recommendations": ["推奨事項1", "推奨事項2"]
  }
}
```

分析結果を統合してください。
