## 指示

以下のポイントに注意して、個々のサブタスク結果を一貫性のある創作物に統合してください:

1. キャラクター、設定、プロット、テーマの一貫性を確保する
2. 物語の流れや構造を最適化する
3. 文体や語調の統一性を維持する
4. 創作全体のバランスと完成度を高める
5. 読者体験を向上させる最終的な調整を行う

## 出力形式

以下のJSON形式で統合結果を出力してください:

```json
{
  "integrated_content": "統合された最終的な創作物",
  "format": "creative_text",
  "metadata": {
    "genre": "ジャンル",
    "theme": "テーマ",
    "key_elements": ["要素1", "要素2"],
    "word_count": 1500
  }
}
```

創作物を統合してください。
